# NexPlay

A mobile-first web app for playing original party and logic games together —
starting with one family, built to scale.

This is the flagship product of the Nex ecosystem. Full context lives in
`docs/`:

- [`docs/INDEX.md`](docs/INDEX.md) — documentation index, start here.
- [`docs/NEXPLAY_PLAN.md`](docs/NEXPLAY_PLAN.md) — vision, scope, and architecture overview.
- [`docs/ROADMAP.md`](docs/ROADMAP.md) — milestone path.
- [`docs/05_engineering/CONVENTIONS.md`](docs/05_engineering/CONVENTIONS.md) — how code in this repo is structured and written.

## Development

Requires [pnpm](https://pnpm.io) (see `docs/00_decisions/architecture/ADR-0001-STACK-AND-PERSISTENCE.md` — pnpm only, no npm/yarn).

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).
